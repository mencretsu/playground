🔑 SiliconFlow API Key Checker

Cek validitas API Key SiliconFlow via endpoint /v1/models




💡 Masukkan API Key lalu klik "Cek Key"
📘 Catatan teknis

- Header Authorization: Bearer {KEY} HARUS ASCII murni.
- Error "Cannot convert value ... to ByteString" terjadi jika ada karakter di luar ASCII (misal: spasi aneh, karakter Unicode, atau line break).
- Form ini akan membersihkan karakter tidak valid sebelum request.
- Status 200 = KEY VALID ✅
- Status 401 = KEY INVALID ❌
