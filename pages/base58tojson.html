🔄 Base58 → JSON Converter

Convert Private Key Base58 (64-byte) ke format yang diterima Phantom
—
—
⚠️ CARA PAKE:
1. Paste private key Base58 (64-byte) dari generator lama lo
2. Klik Convert
3. Copy JSON array yang keluar
4. Import ke Phantom: Settings → Add Wallet → Import Private Key → Paste JSON
💡 Catatan: Converter ini butuh private key 64-byte dalam Base58 (panjang ~88 karakter).
Kalo private key lo cuma 44 karakter (32-byte seed), convertnya bakal gagal.
